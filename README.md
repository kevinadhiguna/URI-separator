# URL / URI separator

This script separates a URL / URI into : 
1. Scheme
2. Authority
3. Query
4. Fragment
5. Path

## How to run :
1) Input the URL / URI you want to separate.
2) Open terminal / command prompt, then run : 
```bash
python uri-separator.py
```

or

```bash
python3 uri-separator.py
```
