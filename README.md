# URI-separator
URI-separator separates URI into : 
1. Scheme
2. Authority
3. Query
4. Fragment
5. Path

## How to run :
1) Input the URI you want to separate.
2) Open terminal / command prompt, then run : 
`python uri-separator.py`
or
`python3 uri-separator.py`
